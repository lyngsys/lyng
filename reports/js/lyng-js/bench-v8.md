# Lyng JS V8 v7 Benchmark Report

This report is generated by `cargo run --release -p lyng-js-bench -- v8suite`.

Each benchmark runs in an isolated `lyng-js --shell` subprocess per sample so
warmup, GC, feedback caches, and tier transitions don't leak between samples.
Score = `100 × reference_µs / mean_µs` (V8 standard formula); higher is better.

## Configuration

- Samples per benchmark: `5`
- Per-sample timeout: `120s`
- lyng-js binary: `target/release/lyng-js`
- V8 v7 sources: `testdata/js-benchmarks/v8-v7`
