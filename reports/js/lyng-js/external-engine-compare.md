# Lyng JS External Engine Comparison

This report is generated by `cargo run --release -p lyng-js-bench -- compare --report reports/js/lyng-js/external-engine-compare.md`.
It runs the same standalone JavaScript workload files through Lyng JS, QuickJS, and Boa.

## Settings

- Report: `reports/js/lyng-js/external-engine-compare.md`
- JSON: `reports/js/lyng-js/external-engine-compare.json`
- Scripts dir: `/tmp/lyng-js-bench-compare-scripts`
- Corpus: `synthetic`
- Full suite: `false`
- Samples: `3`
- Warmup samples: `1`
- Loop trips: `2048`
- Timeout: `30000ms`

## Comparison Policy

- QuickJS is the primary interpreter baseline.
- Boa is a Rust-engine reference point.
- Treat parity as a workload-family measurement, not exact equality across every script.
